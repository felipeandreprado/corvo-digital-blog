# 🌐 Blog Tech

Este é um blog de tecnologia criado com dois objetivos principais:

- 📚 Fixar conhecimento colocando em prática o que aprendo na faculdade e no dia a dia
- 🚀 Explicar tecnologia de forma simples e acessível para iniciantes e entusiastas da tecnlogia.
- 💼 Também funciona como portfólio pessoal de desenvolvimento

---

## 🎯 Objetivo

Projeto criado para aprendizado contínuo, unindo prática e teoria, além de servir como portfólio de evolução como desenvolvedor.
